
# 📊 Customer Call Data Analysis (Data Cleaning)

## 📌 Overview

This project focuses on **data cleaning and exploratory data analysis (EDA)** on a customer call dataset. The main objective is to prepare raw data for analysis by handling inconsistencies, missing values, and structural issues, while extracting meaningful observations from the dataset.

---



## 🧠 Objectives

* Clean and preprocess raw customer call data
* Handle missing values and duplicates
* Standardize and format data properly
* Prepare dataset for further analysis or modeling

---

## 🔧 Tools & Technologies

* **Python**
* **Pandas** – Data manipulation and cleaning
* **Jupyter Notebook** – Analysis environment
* **Excel** – Source data

---

## 📊 Key Steps Performed

### 1️⃣ Data Loading

* Imported dataset from Excel file
* Reviewed dataset structure, column names, and data types

### 2️⃣ Data Cleaning

* Handled missing/null values
* Removed duplicate records
* Fixed inconsistent formats
* Renamed columns for clarity
* Converted data types (e.g., dates, numeric fields)

### 3️⃣ Feature Engineering

* Created new features where required
* Extracted useful information from existing columns



## 📈 Key Observations

* Presence of missing and inconsistent data in raw dataset
* Duplicate entries affecting data quality
* Patterns in customer call records identified through statistical summaries
* Dataset required preprocessing before any advanced analysis or modeling
